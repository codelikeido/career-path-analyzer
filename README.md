# career-path-analyzer
An AI-powered web application that analyzes your skills and suggests personalized career paths, improvements, and growth opportunities.

 Live Demo: https://codelikeido.github.io/career-path-analyzer/

 Overview

The Career Path Analyzer helps users:

Understand suitable career paths based on their skills
Identify skill gaps
Get AI-generated recommendations for improvement

This project is built as a lightweight, browser-based application with no backend required.

Features:
AI-powered career analysis
Skill-based recommendations
Fast and lightweight (runs in browser)
Easily deployable (GitHub Pages, Netlify, Vercel)
Secure (API key is user-provided and not stored)
Tech Stack:
HTML
CSS
JavaScript
Anthropic Claude API
 
How to Run Locally

1.Download or clone this repository:

2.git clone https://github.com/codelikeido/career-path-analyzer.git
Open the project folder

3.Double-click on:

index.html
